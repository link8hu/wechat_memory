# AGENTS.md - Oh My OpenAgent 工作指南

## 🎯 项目：微信聊天记录导出工具

### 当前任务
开发一个开源的微信聊天记录导出工具，基于 GitHub 现有项目，用于生成"两个人的回忆"网站数据。

### 可用 Agent
- **Sisyphus** (Kimi K2.5) - 总协调/产品规划
- **Prometheus** (GLM-4.7) - 需求分析/战略规划  
- **Oracle** (GLM-4.7) - 风险评估/方案审查
- **Explore** (GLM-4.7) - 竞品调研/技术调研
- **Hephaestus** (GLM-4.7) - 技术开发/代码实现
- **Librarian** (GLM-4.7) - 文档整理/知识管理

### 工作流程
1. **启动 Sisyphus** - 总体规划任务分解
2. **并行执行** - Explore 调研 + Prometheus 分析 + Hephaestus 技术预研
3. **合并方案** - Oracle 审查风险
4. **文档输出** - Librarian 整理最终文档

### 启动命令
```bash
# 在项目目录启动
cd ~/.openclaw/workspace/wechat-memory-export
opencode
ultrawork 开发微信聊天记录导出工具
```

### 输出目标
1. 技术调研报告
2. MVP 开发计划
3. 数据库结构文档
4. 导出工具代码
5. 用户使用指南

### 注意事项
- 优先使用现有开源项目
- 支持 macOS 和 Windows
- 导出格式：HTML + JSON
- 保护用户隐私，本地处理
