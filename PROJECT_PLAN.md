# 微信聊天记录导出工具 - 开源项目

## 🎯 项目目标
基于 GitHub 现有开源项目，开发一个简单易用的微信聊天记录导出工具，用于生成"两个人的回忆"网站数据。

## 📚 参考项目
1. **WeChatExporter** (de8ug/WeChatExporter) - iOS 备份导出
2. **WechatExport-IOS** (starymz/WechatExport) - HTML/TXT导出
3. **wechat-export** (xyzibu/wechat-export) - 机器人导出

## 🛠️ 技术栈
- **后端**: Node.js + SQLite
- **前端**: Vue.js / React
- **导出格式**: HTML + JSON
- **支持平台**: macOS / Windows

## 📋 功能清单
- [ ] iTunes 备份解析
